﻿Konwencje designu

﻿header - tag z głównym tematem strony - introductory information or navigation links - ta listwa u góry strony

main - główny content strony  - 1 na stronę

nav - linki nawigacyjne po stronie i podstronach

div - groups content

article - groups standalone content

section - groups related content

footer - stopka

audio - tag zawierający pliki dźwiekowe:

  • <audio controls> <source src = "" type = "audio/mpeg" />  </audio>

figure - tag zawierający jakiś obraz, figcaption - podpis tego obrazu

label - plakietka obok inputów

fieldset - groups a set of radio buttons 

legend - description of the fieldset

time datetime - element zawierający datę np. <time datetime = "20.07.2019"> 20 lipca </time>

ukryty element dla screen readerów - overflow:hidden; position: absolute; top: auto, left: -20000px; width i hight nie mogą być zerowe

w <a> powinien być krótki opis linku

accesskey - dla użytkowników klawiatury    

tabindex - na jaki tag złapie focusa przy kliknięciu tab, tabindex = -1, 0 lub 1,2,3,4,5,6,7,8,9 itd.
